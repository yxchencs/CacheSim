1. 均值、方差和置信区间
均值（Mean）：代表数据的平均水平。观察其是否在一个合理的范围内。
方差（Variance）：度量数据离散程度。较大方差说明数据更分散。
95%置信区间（Confidence Interval）：表示真实均值在此范围内的可能性是 95%。如果置信区间比较窄，说明估计的均值更加可靠。
2. 偏度和峰度
偏度（Skewness）：

正偏度（>0）：数据右偏，长尾在右边。
负偏度（<0）：数据左偏，长尾在左侧。
接近零：对称分布。
峰度（Kurtosis）：

大于3（默认值为Fisher正态分布）：尖峰分布（leptokurtic）。
等于3：正态分布。
小于3：平坦分布（platykurtic）。
3. 正态性检验
Shapiro-Wilk Test：用于评估数据是否来自正态分布。
p-value > 0.05：未拒绝数据符合正态分布假设。
p-value <= 0.05：拒绝数据符合正态分布假设。
4. 数据可视化分析
直方图和KDE：

观察数据的形态（正态、人字型、尖峰、平坦等）。
KDE 曲线平滑度提供分布密度的直观理解。
Q-Q Plot：

比较数据分布与理论正态分布。点沿对角线分布表示数据服从正态分布，否则偏离。
综合分析策略
均值和方差提供数据的集中趋势和变异性。
偏度和峰度帮助理解数据对称性和尖峰性。
正态性检验结合 Q-Q 图共同评估数据与正态分布的吻合程度。
结合直方图和KDE，形象化观测数据分布及其特性。
对于应用场景：

如果数据的偏度或峰度不符合预期，可以考虑数据转换（如对数、平方根等）来调整分布。
在建模中，了解数据的分布情况有助选择合适的统计方法或模型。
